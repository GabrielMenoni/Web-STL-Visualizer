# Web STL Visualizer

A web-based 3D viewer for STL files with real-time transformations and multi-model support. Load, visualize, and manipulate multiple STL files directly in your browser with precise position and rotation controls.

## ‚ú® Application capabilities

- **Multiple File Support** - Load and manage multiple STL files simultaneously
- **Visual Differentiation** - Each model gets a unique color for easy identification
- **Visibility Control** - Toggle individual model visibility on/off
- **Precise Transformations** - Control position (mm, cm, inches) and rotation (degrees) with accuracy
- **Interactive 3D View** - Orbit, zoom, and pan controls for easy navigation
- **Real-time Updates** - See changes immediately as you adjust transformations
- **Unit Conversion** - Support for millimeters, centimeters, and inches with automatic conversion

## üöÄ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/GabrielMenoni/Web-STL-Visualizer.git
cd Web-STL-Visualizer
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
